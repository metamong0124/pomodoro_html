    사용법

    타이머
    • 프리셋 버튼으로 5~60분 중 선택, 또는 직접입력으로 시/분/초 지정
    • −5 / +5 버튼으로 진행 중에도 5분씩 늘리고 줄일 수 있어요
    • 시계 모양 휴식 버튼은 간단한 휴식 타이머를 바로 시작해요
    뽀모도로
    • 설정 → 뽀모도로 시작 버튼으로 집중·휴식 사이클 시작
    • 할 일의 ▼ 드롭다운 → 뽀모도로 시작으로 특정 할 일에 집중 시간 할당
    • 4회 완료 후에는 긴 휴식이 자동으로 들어가요
    할 일 태그
    • 연필 아이콘(✎)을 누르면 편집창이 열려요. 마감과 반복 줄에서 원하는 칩을 눌러 지정
    • 마감일 "날짜…" 칩을 누르면 달력이 열려 직접 선택 가능
    • 빠른 입력: 추가할 때 회의 !내일 @매주처럼 !마감 @반복 태그를 섞어 써도 돼요
    • 태그 예시: !오늘 / !내일 / !모레 / !12/25 / !2026-04-20, @매일 / @매주 / @매월
    할 일 관리
    • 연필 아이콘(✎)으로 내용 수정 + 마감/반복 시각적 지정
    • 번호(원)를 드래그해서 순서 변경
    • + 버튼으로 하위 할 일 추가
    • 완료한 할 일은 7일 후 자동 삭제 (설정에서 내보내기 가능)
    • 우측 상단 눈 아이콘으로 할 일을 가릴 수 있어요
    메모
    • 상단 메모 탭에서 제목과 내용을 작성하고 저장 버튼으로 저장
    • 목록 버튼으로 저장된 메모를 최근 수정순으로 확인 (검색 가능)
    • 글씨 크기 버튼으로 작게/보통/크게 전환
    • 글머리 버튼 → 불렛(•), 원숫자(①), 숫자(1.) 중 선택해 줄 앞에 일괄 적용
    • 여러 줄을 선택한 뒤 적용하면 각 줄에 순번이 붙어요
    • 복사 버튼으로 본문 전체를 클립보드에 복사
    • 작성 중인 내용은 자동으로 임시 저장돼요
    키보드 단축키
    • Space — 타이머 시작/일시정지 (입력창 포커스 아닐 때)
    • / — 할 일 입력창으로 포커스 이동
    • Esc — 설정·도움말 창 닫기, 할 일 편집 취소
    • Enter — 할 일 추가, 편집 저장
    기타
    • 상단 메시지는 클릭해서 수정 가능
    • 테마 동그라미로 색상 변경, +로 추가 테마 보기
    • 통계 탭의 고급 설정에서 무음 모드, 통계 기간, 주 시작 요일 변경
    • 데이터는 모두 이 브라우저에만 저장돼요 (외부 전송 없음)
